Stock Analysis Processor
This is a Streamlit application that processes financial statements (Income Statement, Balance Sheet, Cash Flow) into a proprietary Excel model.

How to Deploy to Streamlit Cloud
	1.	Push this repository to GitHub. 
	2.	Go to share.streamlit.io. 
	3.	Connect your GitHub account. 
	4.	Select this repository and the streamlit_app.py file. 
	5.	Click Deploy. 
Usage
	1.	Upload the three required CSV files (TICKER_annual_financials.csv, TICKER_annual_balance-sheet.csv, TICKER_annual_cash-flow.csv). 
	2.	Click Generate Analysis. 
	3.	Preview the data in the tabs or download the full Excel report with formulas and formatting.
